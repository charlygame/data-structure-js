# data-structure-js

使用javascript 实现比常见的数据结构。
```

|- .vscode                          [vscode运行调试文件]
|- modules                          [数据结构根目录]
|- |-find                           [查找相关实现]
|- |-sort                           [排序相关]
|- |-sort-bubble.js                 [冒泡排序的实现]
|- |-sort-quick.js                  [快速排序的实现]
|- index.js                         [入口文件]

```
